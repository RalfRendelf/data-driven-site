<h2>std::partition_copy: Копирование элементов в два диапазона по предикату</h2>

<p>
<strong>`std::partition_copy`</strong> — алгоритм из библиотеки <code>&lt;algorithm&gt;</code>, который копирует элементы диапазона в два разных диапазона в зависимости от результата предиката.
Работает за линейное время <strong>O(N)</strong> и использует <strong>O(N)</strong> дополнительной памяти (для двух выходных диапазонов).
</p>

<h4>Параметры</h4>
<table>
  <tr>
    <th>Параметр</th>
    <th>Описание</th>
  </tr>
  <tr>
    <td><code>first, last</code></td>
    <td>Диапазон элементов</td>
  </tr>
  <tr>
    <td><code>out_true</code></td>
    <td>Итератор на начало диапазона для элементов, удовлетворяющих предикату</td>
  </tr>
  <tr>
    <td><code>out_false</code></td>
    <td>Итератор на начало диапазона для элементов, не удовлетворяющих предикату</td>
  </tr>
  <tr>
    <td><code>pred</code></td>
    <td>Унарный предикат для разделения</td>
  </tr>
</table>

<h4>Возвращаемое значение</h4>
<p>
Возвращает <code>std::pair</code> из итераторов на элементы, следующие за последними скопированными элементами в каждом из выходных диапазонов.
</p>

<h3>Заключение</h3>
<p>
<strong>`std::partition_copy`</strong> — алгоритм для копирования элементов диапазона в два разных диапазона по заданному предикату.
Полезен, когда нужно разделить элементы без изменения исходного диапазона.
</p>
