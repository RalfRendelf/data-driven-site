<h2>Binary Search (Бинарный поиск)</h2>

<p><strong>Категория:</strong> Поиск</p>

<h3>Описание</h3>
<p>Бинарный поиск - это алгоритм поиска позиции целевого значения в отсортированном массиве. Алгоритм работает методом "разделяй и властвуй", сравнивая целевое значение со средним элементом массива и, в зависимости от результата, продолжает поиск в левой или правой половине.</p>

<h3>Принцип работы</h3>
<ol>
<li>Сравнить целевое значение со средним элементом массива</li>
<li>Если они равны, вернуть индекс элемента</li>
<li>Если целевое значение меньше среднего, повторить поиск в левой половине</li>
<li>Если целевое значение больше среднего, повторить поиск в правой половине</li>
<li>Продолжать до тех пор, пока не найдется элемент или диапазон не будет исчерпан</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(log n)</li>
<li>Пространственная сложность: O(1) для итеративной реализации, O(log n) для рекурсивной</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Высокая эффективность при работе с отсортированными данными</li>
<li>Меньшее количество сравнений по сравнению с линейным поиском</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует предварительной сортировки данных</li>
<li>Менее эффективен для часто изменяющихся наборов данных</li>
</ul>

<h3>Примеры реализации</h3>

<h4>Python</h4>
<pre>
<code>
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
            
    return -1  # элемент не найден

# Рекурсивная реализация
def binary_search_recursive(arr, target, left, right):
    if left > right:
        return -1  # элемент не найден
    
    mid = (left + right) // 2
    
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_recursive(arr, target, mid + 1, right)
    else:
        return binary_search_recursive(arr, target, left, mid - 1)
</code>
</pre>

function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;
  
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    
    if (arr[mid] === target) {
      return mid;
    }
    
    if (arr[mid] < target) {
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }
  
  return -1; // элемент не найден
}

// Рекурсивная реализация
function binarySearchRecursive(arr, target, left = 0, right = arr.length - 1) {
  if (left > right) {
    return -1; // элемент не найден
  }
  
  const mid = Math.floor((left + right) / 2);
  
  if (arr[mid] === target) {
    return mid;
  } else if (arr[mid] < target) {
    return binarySearchRecursive(arr, target, mid + 1, right);
  } else {
    return binarySearchRecursive(arr, target, left, mid - 1);
  }
}

    int left = 0, right = size - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;  // избегаем переполнения
