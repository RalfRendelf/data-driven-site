<h2>Dijkstra (Алгоритм Дейкстры)</h2>

<p><strong>Категория:</strong> Графы</p>

<h3>Описание</h3>
<p>Алгоритм Дейкстры - это алгоритм поиска кратчайших путей от одной вершины (источника) ко всем другим вершинам взвешенного графа с неотрицательными весами. Был разработан голландским информатиком Эдсгером Дейкстрой в 1956 году.</p>

<h3>Принцип работы</h3>
<ol>
<li>Инициализировать расстояния от источника до всех вершин как бесконечность, кроме источника (0)</li>
<li>Создать приоритетную очередь (минимальную кучу) с вершинами, отсортированными по расстоянию</li>
<li>Пока очередь не пуста, выбрать вершину с минимальным расстоянием и рассмотреть всех ее соседей</li>
<li>Если новый путь к соседу короче известного, обновить расстояние</li>
<li>Повторять, пока все вершины не будут обработаны</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность зависит от реализации:
  <ul>
    <li>Наивная реализация с массивом: O(V²)</li>
    <li>С бинарной кучей: O((V + E) log V)</li>
    <li>С фибоначчиевой кучей: O(V log V + E)</li>
  </ul>
</li>
<li>Пространственная сложность:
  <ul>
    <li>Базовая: O(V) для хранения расстояний</li>
    <li>Дополнительно O(V) при использовании приоритетной очереди</li>
    <li>Всего: O(V + E) в худшем случае</li>
  </ul>
</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Гарантированно находит кратчайший путь в графах с неотрицательными весами</li>
<li>Относительно прост в реализации</li>
<li>Широко применяется в сетевых протоколах и GPS-навигации</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Не работает с отрицательными весами ребер</li>
<li>Может быть неэффективным для плотных графов</li>
<li>Требует знания всей топологии графа заранее</li>
</ul>

<h3>Пример использования</h3>
<pre>
<code>
function dijkstra(graph, start) {
  const distances = {};
  const previous = {};
  const unvisited = new Set();
  
  // Инициализация
  for (const vertex in graph) {
    distances[vertex] = Infinity;
    previous[vertex] = null;
    unvisited.add(vertex);
  }
  distances[start] = 0;
  
  while (unvisited.size > 0) {
    // Находим вершину с минимальным расстоянием
    let current = null;
    let minDistance = Infinity;
    
    for (const vertex of unvisited) {
      if (distances[vertex] < minDistance) {
        minDistance = distances[vertex];
        current = vertex;
      }
    }
    
    // Если расстояние равно Infinity, то остальные вершины недостижимы
    if (distances[current] === Infinity) break;
    
    // Обновляем расстояния для соседей
    for (const neighbor in graph[current]) {
      const alt = distances[current] + graph[current][neighbor];
      if (alt < distances[neighbor]) {
        distances[neighbor] = alt;
        previous[neighbor] = current;
      }
    }
    
    unvisited.delete(current);
  }
  
  return distances;
}
