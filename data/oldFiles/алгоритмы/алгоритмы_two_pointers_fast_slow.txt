<h2>Two Pointers: Fast-slow (Два указателя: быстрый-медленный)</h2>

<p><strong>Категория:</strong> Массивы/Связные списки</p>

<h3>Описание</h3>
<p>Алгоритм двух указателей, когда один указатель (быстрый) движется быстрее другого (медленного). Наиболее часто используется для обнаружения циклов в связных списках или массивах, а также для решения задач на нахождение середины списка.</p>

<h3>Принцип работы</h3>
<ol>
<li>Инициализировать два указателя в начальной позиции</li>
<li>Двигать медленный указатель на 1 шаг, быстрый - на 2 шага (или в n раз быстрее)</li>
<li>При наличии цикла указатели в конечном итоге встретятся</li>
<li>При поиске середины: когда быстрый достигнет конца, медленный будет в середине</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(n), где n - длина массива или списка</li>
<li>Пространственная сложность: O(1) - только для дополнительных переменных</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Обнаружение циклов без дополнительной памяти</li>
<li>Эффективное нахождение середины списка без предварительного подсчета длины</li>
<li>Минимальное использование памяти</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Менее интуитивно понятен</li>
<li>Применим только к специфическим задачам</li>
</ul>

<h3>Пример на C++: Обнаружение цикла в связном списке</h3>
<pre>
<code>
#include &lt;iostream&gt;

struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class Solution {
public:
    bool hasCycle(ListNode *head) {
        if (!head || !head->next) {
            return false;
        }
        
        ListNode *slow = head;
        ListNode *fast = head->next;
        
        while (slow != fast) {
            if (!fast || !fast->next) {
                return false; // Достигли конца списка
            }
            
            slow = slow->next;      // Медленный указатель - 1 шаг
            fast = fast->next->next; // Быстрый указатель - 2 шага
        }
        
        return true; // Указатели встретились - есть цикл
    }
    
    // Нахождение середины связного списка
    ListNode* middleNode(ListNode* head) {
        ListNode *slow = head;
        ListNode *fast = head;
        
        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }
        
        return slow; // slow теперь указывает на середину
    }
};

int main() {
    // Создаем список: 1 -> 2 -> 3 -> 4 -> 5
    ListNode* head = new ListNode(1);
    head->next = new ListNode(2);
    head->next->next = new ListNode(3);
    head->next->next->next = new ListNode(4);
    head->next->next->next->next = new ListNode(5);
    
    Solution solution;
    
    std::cout << "Middle node value: " << solution.middleNode(head)->val << std::endl;
    
    // Создаем цикл: 5 -> 3
    head->next->next->next->next->next = head->next->next;
    
    if (solution.hasCycle(head)) {
        std::cout << "Cycle detected in the linked list." << std::endl;
    } else {
        std::cout << "No cycle detected." << std::endl;
    }
    
    return 0;
}
</code>
</pre>

<h3>Пример на C: Нахождение дубликата числа в массиве</h3>
<pre>
<code>
#include &lt;stdio.h&gt;

// В массиве длиной n+1 все числа от 1 до n, одно число повторяется
// Используем массив как связный список, где значение указывает на индекс
int findDuplicate(int nums[], int size) {
    // Шаг 1: Найдем точку встречи в цикле
    int slow = nums[0];
    int fast = nums[0];
    
    // Двигаем slow на 1 шаг, fast на 2 шага
    do {
        slow = nums[slow];
        fast = nums[nums[fast]];
    } while (slow != fast);
    
    // Шаг 2: Найдем начало цикла
    slow = nums[0];
    while (slow != fast) {
        slow = nums[slow];
        fast = nums[fast];
    }
    
    return slow;
}

int main() {
    int nums[] = {1, 3, 4, 2, 2}; // 2 повторяется
    int size = sizeof(nums) / sizeof(nums[0]);
    
    int duplicate = findDuplicate(nums, size);
    
    printf("Duplicate number: %d\n", duplicate);
    
    return 0;
}
