<h3>Алгоритмы для работы с побитовыми операциями</h3>
<p>В этом разделе представлены эффективные алгоритмы для работы с побитовыми операциями.</p>

<h4>Реверс битов</h4>
<p>Функция, которая принимает на вход 32-битное беззнаковое целое число (uint32_t) и возвращает число, в котором порядок битов изменен на обратный:</p>

<pre><code>uint32_t reverseBits(uint32_t n) {
    uint32_t result = 0;
    for (int i = 0; i < 32; i++) {
        result <<= 1;       // Освобождаем место под новый бит
        result |= (n & 1);  // Записываем младший бит из n
        n >>= 1;            // Убираем уже обработанный бит из n
    }
    return result;
}</code></pre>
<p><strong>Сложность по времени:</strong> O(1) - фиксированное количество итераций<br>
<strong>Сложность по памяти:</strong> O(1) - используется константное количество дополнительной памяти</p>

<h4>Оптимизированный реверс битов</h4>
<p>Алгоритм с делением пополам, работающий быстрее за счет параллельной обработки групп битов:</p>

<pre><code>uint32_t reverseBits(uint32_t n) {
    n = ((n & 0xffff0000) >> 16) | ((n & 0x0000ffff) << 16);
    n = ((n & 0xff00ff00) >> 8)  | ((n & 0x00ff00ff) << 8);
    n = ((n & 0xf0f0f0f0) >> 4)  | ((n & 0x0f0f0f0f) << 4);
    n = ((n & 0xcccccccc) >> 2)  | ((n & 0x33333333) << 2);
    n = ((n & 0xaaaaaaaa) >> 1)  | ((n & 0x55555555) << 1);
    return n;
}</code></pre>
<p><strong>Сложность по времени:</strong> O(1) - фиксированное количество операций<br>
<strong>Сложность по памяти:</strong> O(1) - используется константное количество дополнительной памяти</p>

<h4>Другие полезные побитовые операции</h4>
<p>Проверка, является ли число степенью двойки:</p>
<pre><code>bool isPowerOfTwo(uint32_t n) {
    return n && (n & (n - 1));
}</code></pre>
<p><strong>Сложность по времени:</strong> O(1)<br>
<strong>Сложность по памяти:</strong> O(1)</p>

<p>Подсчет единичных битов (population count):</p>
<pre><code>int countSetBits(uint32_t n) {
    int count = 0;
    while (n) {
        count++;
        n &= (n - 1);  // Удаляем младший единичный бит
    }
    return count;
}</code></pre>
<p><strong>Сложность по времени:</strong> O(k), где k - количество установленных битов<br>
<strong>Сложность по памяти:</strong> O(1)</p>

<p>Проверка, установлен ли i-й бит:</p>
<pre><code>bool isBitSet(uint32_t n, int i) {
    return (n & (1 << i)) != 0;
}</code></pre>
<p><strong>Сложность по времени:</strong> O(1)<br>
<strong>Сложность по памяти:</strong> O(1)</p>
