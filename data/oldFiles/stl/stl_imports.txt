<h2>Импорт библиотек</h2>

<p><strong>Категория:</strong> Модули</p>

<h3>Описание</h3>
<p>В C++ импорт библиотек осуществляется с помощью директивы #include, которая включает содержимое заголовочного файла в исходный код. С появлением C++20 также доступна поддержка модулей, которая предоставляет более современный способ организации кода.</p>

<h3>Добавлен в стандарт:</h3>
<p>#include - с самого начала C++, модули - с C++20</p>

<h3>Основные методы:</h3>
<p>В этом разделе представлены основные способы импорта библиотек и их использование:</p>

<h4>C++</h4>
<pre>
<code>
// =======================
// Примеры импорта библиотек в C++
// =======================

/*

СТАНДАРТНЫЕ ЗАГОЛОВКИ C
*/

#include &lt;stdio.h&gt;       // Ввод-вывод
#include &lt;stdlib.h&gt;      // Стандартные утилиты
#include &lt;string.h&gt;      // Работа со строками
#include &lt;math.h&gt;        // Математические функции
#include &lt;time.h&gt;        // Работа со временем
#include &lt;ctype.h&gt;       // Классификация символов
#include &lt;assert.h&gt;      // Проверки
#include &lt;errno.h&gt;       // Коды ошибок
#include &lt;limits.h&gt;      // Пределы типов
#include &lt;float.h&gt;       // Пределы вещественных типов

/*

СТАНДАРТНЫЕ ЗАГОЛОВКИ C++
*/

#include &lt;iostream&gt;
#include &lt;vector&gt;
#include &lt;memory&gt;
#include &lt;algorithm&gt;

/**

Назначение: вывести строку в стандартный поток вывода.

Параметры:

строковый литерал
*/
// std::cout — выводим текст в консоль
void example_iostream() {
std::cout &lt;&lt; "Пример использования стандартных библиотек" &lt;&lt; std::endl;
}

/**

Назначение: создать контейнер и отсортировать его.

Параметры:

нет
*/
// std::vector + std::sort — создаём массив и сортируем
void example_vector_sort() {
std::vector&lt;int&gt; numbers = {5, 2, 8, 1, 9};
std::sort(numbers.begin(), numbers.end());

std::cout &lt;&lt; "Отсортированные числа: ";
for (const auto&amp; num : numbers) {
std::cout &lt;&lt; num &lt;&lt; " ";
}
std::cout &lt;&lt; std::endl;
}

/**

Назначение: создать умный указатель и получить значение.

Параметры:

нет
*/
// std::unique_ptr — создаём объект в динамической памяти
void example_unique_ptr() {
std::unique_ptr&lt;int&gt; ptr = std::make_unique&lt;int&gt;(42);
std::cout &lt;&lt; "Значение через unique_ptr: " &lt;&lt; *ptr &lt;&lt; std::endl;
}

int main() {
example_iostream();
example_vector_sort();
example_unique_ptr();
return 0;
}
