<h2>std::mutex</h2>

<p><strong>Категория:</strong> Многопоточность</p>

<h3>Описание</h3>
<p>std::mutex - это примитив синхронизации, который используется для контроля доступа к общим данным из нескольких потоков. Он обеспечивает взаимное исключение, позволяя только одному потоку за раз захватывать мьютекс.</p>

<h3>Добавлен в стандарт:</h3>
<p>C++11</p>

<h3>Пример использования в C++</h3>
<p>Вот пример использования std::mutex в C++:</p>

<pre>
<code>
// Пример использования std::mutex на C++

#include &lt;iostream&gt;
#include &lt;thread&gt;
#include &lt;mutex&gt;
#include &lt;vector&gt;

std::mutex mtx; // Общий мьютекс
int counter = 0;

void worker_function(int id, int iterations) {
    for (int i = 0; i < iterations; ++i) {
        // Блокировка мьютекса
        mtx.lock();
        counter++;
        std::cout << "Thread " << id << " incremented counter to: " << counter << std::endl;
        mtx.unlock(); // Разблокировка
        
        // Небольшая задержка
        std::this_thread::sleep_for(std::chrono::milliseconds(10));
    }
}

int main() {
    std::vector&lt;std::thread&gt; threads;
    
    // Создание 5 потоков
    for (int i = 0; i < 5; ++i) {
        threads.emplace_back(worker_function, i, 3);
    }
    
    // Ожидание завершения всех потоков
    for (auto&amp; t : threads) {
        t.join();
    }
    
    std::cout << "Final counter value: " << counter << std::endl;
    
    return 0;
}
</code>
</pre>

<h3>Реализация на C</h3>
<p>Для сравнения, вот как можно реализовать аналогичную функциональность на C с использованием pthread мьютексов:</p>

<pre>
<code>
/***
 * Для компиляции необходима поддержка POSIX threads
 * Компиляция: gcc -o mutex_example mutex_example.c -lpthread
 ***/

#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;pthread.h&gt;
#include &lt;unistd.h&gt;

pthread_mutex_t mtx = PTHREAD_MUTEX_INITIALIZER; // Общий мьютекс
int counter = 0;

// Структура для передачи аргументов в поток
typedef struct {
    int id;
    int iterations;
} ThreadArgs;

// Функция, которая будет выполняться в потоке
void* worker_function(void* args) {
    ThreadArgs* thread_args = (ThreadArgs*)args;
    
    for (int i = 0; i < thread_args->iterations; i++) {
        // Блокировка мьютекса
        pthread_mutex_lock(&mtx);
        counter++;
        printf("Thread %d incremented counter to: %d\n", thread_args->id, counter);
        pthread_mutex_unlock(&mtx); // Разблокировка
        
        // Небольшая задержка
        usleep(10000); // 10 мс
    }
    
    return NULL;
}

// Пример использования
int main() {
    pthread_t threads[5];
    ThreadArgs args[5];
    
    // Создание 5 потоков
    for (int i = 0; i < 5; i++) {
        args[i].id = i;
        args[i].iterations = 3;
        pthread_create(&threads[i], NULL, worker_function, &args[i]);
    }
    
    // Ожидание завершения всех потоков
    for (int i = 0; i < 5; i++) {
        pthread_join(threads[i], NULL);
    }
    
    printf("Final counter value: %d\n", counter);
    
    // Уничтожение мьютекса
    pthread_mutex_destroy(&mtx);
    
    return 0;
}
</code>
