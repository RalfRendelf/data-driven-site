<h2>std::tuple</h2>

<p><strong>Категория:</strong> Контейнеры</p>

<h3>Описание</h3>
<p>std::tuple - это класс шаблона, который представляет собой контейнер фиксированного размера, способный хранить несколько значений разных типов. Он является обобщением std::pair, позволяя хранить произвольное количество значений разных типов.</p>

<h3>Добавлен в стандарт:</h3>
<p>C++11</p>

<h3>Основные методы:</h3>
<p>В этом разделе представлены основные методы std::tuple:</p>

<pre>
<code>
// Пример использования std::tuple на C++

#include &lt;iostream&gt;
#include &lt;tuple&gt;
#include &lt;string&gt;

int main() {
    // Создание tuple
    std::tuple&lt;int, std::string, double&gt; t1(10, "Example", 3.14);
    
    // Создание с помощью make_tuple
    auto t2 = std::make_tuple(20, std::string("Test"), 2.71);
    
    // Доступ к элементам через std::get
    std::cout << "First element: " << std::get&lt;0&gt;(t1) << std::endl;
    std::cout << "Second element: " << std::get&lt;1&gt;(t1) << std::endl;
    std::cout << "Third element: " << std::get&lt;2&gt;(t1) << std::endl;
    
    // Доступ к элементам по типу (если типы уникальны)
    std::cout << "Double element: " << std::get&lt;double&gt;(t1) << std::endl;
    
    // Использование tie для разбора tuple
    int a;
    std::string b;
    double c;
    std::tie(a, b, c) = t1;
    std::cout << "Tied values: " << a << ", " << b << ", " << c << std::endl;
    
    // Изменение значений через get
    std::get&lt;0&gt;(t1) = 100;
    std::cout << "Modified first element: " << std::get&lt;0&gt;(t1) << std::endl;
    
    // Размер tuple
    constexpr size_t size = std::tuple_size_v&lt;decltype(t1)&gt;;
    std::cout << "Tuple size: " << size << std::endl;
    
    // Structured bindings (C++17)
#ifdef __cplusplus >= 201703L
    auto [x, y, z] = t2;
    std::cout << "Structured binding: " << x << ", " << y << ", " << z << std::endl;
#endif
    
    return 0;
}
</code>
</pre>

<h3>Реализация на C:</h3>
<p>Для сравнения, вот как можно реализовать аналогичную функциональность на C:</p>

<pre>
<code>
/***
 * Для компиляции необходима поддержка стандартной библиотеки C
 * Компиляция: gcc -o tuple_example tuple_example.c
 ***/

#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;
#include &lt;string.h&gt;

// Структура, аналогичная std::tuple с тремя полями разных типов
typedef struct {
    int first;
    char* second;
    double third;
} TupleIntStringDouble;

// Функция для создания tuple
TupleIntStringDouble create_tuple(int first, const char* second, double third) {
    TupleIntStringDouble t;
    t.first = first;
    t.third = third;
    t.second = malloc(strlen(second) + 1);
    strcpy(t.second, second);
    return t;
}

// Функция для освобождения памяти
void destroy_tuple(TupleIntStringDouble* t) {
    if (t && t->second) {
        free(t->second);
        t->second = NULL;
    }
}

// Функция для получения первого элемента
int get_first(TupleIntStringDouble* t) {
    return t ? t->first : 0;
}

// Функция для получения второго элемента
char* get_second(TupleIntStringDouble* t) {
    return t ? t->second : NULL;
}

// Функция для получения третьего элемента
double get_third(TupleIntStringDouble* t) {
    return t ? t->third : 0.0;
}

// Функция для изменения первого элемента
void set_first(TupleIntStringDouble* t, int value) {
    if (t) t->first = value;
}

// Функция для изменения второго элемента
void set_second(TupleIntStringDouble* t, const char* value) {
    if (t && t->second) {
        free(t->second);
        t->second = malloc(strlen(value) + 1);
        strcpy(t->second, value);
    }
}

// Функция для изменения третьего элемента
void set_third(TupleIntStringDouble* t, double value) {
    if (t) t->third = value;
}

// Пример использования
int main() {
    // Создание tuple
    TupleIntStringDouble t1 = create_tuple(10, "Example", 3.14);
    
    // Доступ к элементам
    printf("First element: %d\n", get_first(&t1));
    printf("Second element: %s\n", get_second(&t1));
    printf("Third element: %f\n", get_third(&t1));
    
    // Изменение значений
    set_first(&t1, 100);
    printf("Modified first element: %d\n", get_first(&t1));
    
    // Освобождение памяти
    destroy_tuple(&t1);
    
    return 0;
}
</code>
