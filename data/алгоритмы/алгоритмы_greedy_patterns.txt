<h2>Greedy Patterns (Жадные паттерны)</h2>

<p><strong>Категория:</strong> Жадные алгоритмы</p>

<h3>Описание</h3>
<p>Жадные алгоритмы - это алгоритмы, которые строят решение пошагово, на каждом шаге выбирая наиболее выгодный вариант на данный момент. Хотя такие алгоритмы не всегда приводят к глобально оптимальному решению, для многих задач они действительно находят оптимальное решение.</p>

<h3>Основные паттерны жадных алгоритмов</h3>
<ol>
<li>Выбор минимального/максимального элемента</li>
<li>Алгоритм Хаффмана (оптимальное кодирование)</li>
<li>Алгоритм Дейкстры (кратчайший путь)</li>
<li>Алгоритм Прима/Крускала (минимальное остовное дерево)</li>
<li>Задача о непересекающихся отрезках</li>
<li>Задача о рюкзаке с дробными значениями</li>
<li>Планирование задач с минимальной задержкой</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность зависит от конкретного алгоритма (обычно O(n log n) или O(n))</li>
<li>Пространственная сложность: O(1) или O(n) в зависимости от задачи</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Простота реализации</li>
<li>Высокая эффективность по времени</li>
<li>Интуитивно понятный подход</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Не гарантируется оптимальное решение для всех задач</li>
<li>Требуется доказательство корректности для каждой задачи</li>
<li>Может быть трудно определить, подходит ли задача для жадного подхода</li>
</ul>

<h3>Пример на C++: Задача о непересекающихся отрезках</h3>
<pre>
<code>
#include &lt;iostream&gt;
#include &lt;vector&gt;
#include &lt;algorithm&gt;

struct Interval {
    int start, end;
    Interval(int s, int e) : start(s), end(e) {}
    
    bool operator<(const Interval& other) const {
        return end < other.end;
    }
};

// Находит максимальное количество непересекающихся отрезков
std::vector&lt;Interval&gt; maxNonOverlappingIntervals(std::vector&lt;Interval&gt; intervals) {
    if (intervals.empty()) return {};
    
    // Сортируем по концам отрезков
    std::sort(intervals.begin(), intervals.end());
    
    std::vector&lt;Interval&gt; result;
    result.push_back(intervals[0]);
    int lastEnd = intervals[0].end;
    
    for (int i = 1; i < intervals.size(); i++) {
        // Если начало текущего отрезка >= концу последнего выбранного
        if (intervals[i].start >= lastEnd) {
            result.push_back(intervals[i]);
            lastEnd = intervals[i].end;
        }
    }
    
    return result;
}

int main() {
    std::vector&lt;Interval&gt; intervals = {
        {1, 4}, {3, 5}, {0, 6}, {5, 7}, {8, 9}, {5, 9}
    };
    
    auto result = maxNonOverlappingIntervals(intervals);
    
    std::cout << "Непересекающиеся отрезки:" << std::endl;
    for (const auto& interval : result) {
        std::cout << "[" << interval.start << ", " << interval.end << "]" << std::endl;
    }
    
    return 0;
}
</code>
</pre>

<h3>Пример на C: Задача о рюкзаке с дробными значениями</h3>
<pre>
<code>
#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;

typedef struct {
    int value;
    int weight;
    double ratio;
} Item;

// Функция сравнения для сортировки по убыванию отношения стоимости к весу
int compare(const void* a, const void* b) {
    Item* itemA = (Item*)a;
    Item* itemB = (Item*)b;
    
    if (itemB->ratio > itemA->ratio) return 1;
    else if (itemB->ratio < itemA->ratio) return -1;
    return 0;
}

double fractionalKnapsack(Item items[], int n, int capacity) {
    // Вычисляем отношение стоимости к весу для каждого элемента
    for (int i = 0; i < n; i++) {
        items[i].ratio = (double)items[i].value / items[i].weight;
    }
    
    // Сортируем элементы по отношению стоимости к весу (по убыванию)
    qsort(items, n, sizeof(Item), compare);
    
    double totalValue = 0.0;
    int currentWeight = 0;
    
    for (int i = 0; i < n; i++) {
        // Если можем добавить целиком
        if (currentWeight + items[i].weight <= capacity) {
            currentWeight += items[i].weight;
            totalValue += items[i].value;
        } else {
            // Добавляем часть элемента
            int remainingCapacity = capacity - currentWeight;
            totalValue += items[i].ratio * remainingCapacity;
            break;
        }
    }
    
    return totalValue;
}

int main() {
    Item items[] = {
        {60, 10, 0}, {100, 20, 0}, {120, 30, 0}
    };
    int n = sizeof(items) / sizeof(items[0]);
    int capacity = 50;
    
    double maxValue = fractionalKnapsack(items, n, capacity);
    
    printf("Максимальная стоимость, которую можно получить: %.2f\n", maxValue);
    
    return 0;
}
</code>
