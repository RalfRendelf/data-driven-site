<h2>KMP (Knuth-Morris-Pratt Algorithm)</h2>

<p><strong>Категория:</strong> Поиск</p>

<h3>Описание</h3>
<p>Алгоритм Кнута-Морриса-Пратта (KMP) - это алгоритм поиска подстроки, который ищет вхождения образца (подстроки) в текст (строку) с высокой эффективностью. В отличие от наивного подхода, KMP использует информацию о совпадающих символах для предотвращения ненужных сравнений.</p>

<h3>Принцип работы</h3>
<ol>
<li>Построить таблицу частичных совпадений (failure function или LPS - Longest Proper Prefix which is also Suffix) для образца</li>
<li>Сравнивать символы образца с символами текста слева направо</li>
<li>Если символы не совпадают, использовать таблицу LPS, чтобы определить, сколько символов уже совпадают и пропустить ненужные сравнения</li>
<li>Продолжать, пока не найдутся все вхождения или текст не будет полностью пройден</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(n + m), где n - длина текста, m - длина образца. Алгоритм достигает линейной сложности за счет эффективного использования информации о предыдущих совпадениях.</li>
<li>Пространственная сложность: O(m) для хранения таблицы LPS (Longest Proper Prefix), которая позволяет избежать повторных сравнений в тексте.</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Гарантированная линейная временная сложность</li>
<li>Нет необходимости в откате указателя текста</li>
<li>Эффективен для поиска длинных образцов с повторяющимися подстроками</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует дополнительной памяти для таблицы LPS</li>
<li>Сложнее реализовать по сравнению с наивным алгоритмом</li>
<li>Не всегда эффективен для коротких образцов</li>
</ul>

function computeLPS(pattern) {
  const m = pattern.length;
  const lps = new Array(m).fill(0);
  let len = 0; // длина предыдущего собственного префикса
  let i = 1;
  
  while (i < m) {
    if (pattern[i] === pattern[len]) {
      len++;
      lps[i] = len;
      i++;
    } else {
      if (len !== 0) {
        len = lps[len - 1]; // не увеличиваем i, потому что не знаем, как сравнивать с pattern[i]
      } else {
        lps[i] = 0;
        i++;
      }
    }
  }
  
  return lps;
}

function KMPSearch(text, pattern) {
  const n = text.length;
  const m = pattern.length;
  
  if (m === 0) return [];
  
  // Получить LPS массив
  const lps = computeLPS(pattern);
  
  const result = [];
  let i = 0; // индекс для text
  let j = 0; // индекс для pattern
  
  while (i < n) {
    if (pattern[j] === text[i]) {
      i++;
      j++;
    }
    
    if (j === m) {
      result.push(i - j); // найдено совпадение
      j = lps[j - 1];
    } else if (i < n && pattern[j] !== text[i]) {
      if (j !== 0) {
        j = lps[j - 1];
      } else {
        i++;
      }
    }
  }
  
  return result;
}

// Пример использования:
const text = "ABABDABACDABABCABCABCABCABC";
const pattern = "ABABCABCABCABCABC";
console.log(KMPSearch(text, pattern)); // выводит индексы вхождений
