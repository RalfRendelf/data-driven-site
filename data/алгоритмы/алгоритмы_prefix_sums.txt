<h2>Prefix Sums (Префиксные суммы)</h2>

<p><strong>Категория:</strong> Массивы</p>

<h3>Описание</h3>
<p>Префиксные суммы - это структура данных, позволяющая за O(1) вычислять сумму на любом подотрезке массива после O(n) предварительной обработки. Префиксная сумма для позиции i - это сумма всех элементов массива от начала до i-й позиции включительно.</p>

<h3>Принцип работы</h3>
<ol>
<li>Построить массив префиксных сумм: prefix[i] = prefix[i-1] + arr[i]</li>
<li>Для запроса суммы на подотрезке [l, r]: sum(l, r) = prefix[r] - prefix[l-1]</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность построения: O(n)</li>
<li>Временная сложность запроса: O(1)</li>
<li>Пространственная сложность: O(n)</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Очень быстрые запросы суммы на подотрезке</li>
<li>Простота реализации</li>
<li>Можно модифицировать для других операций (например, XOR, произведение)</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Не поддерживают обновления элементов</li>
<li>Требуют дополнительной памяти</li>
<li>Не подходят для операций, не обладающих свойством инвертируемости</li>
</ul>

<h3>Пример на C++</h3>
<pre>
<code>
#include &lt;iostream&gt;
#include &lt;vector&gt;

class PrefixSum {
private:
    std::vector&lt;long long&gt; prefix;
public:
    PrefixSum(const std::vector&lt;int&gt;&amp; arr) {
        prefix.resize(arr.size() + 1, 0);
        for (int i = 0; i &lt; arr.size(); i++) {
            prefix[i + 1] = prefix[i] + arr[i];
        }
    }
    
    long long getSum(int l, int r) {
        // Возвращает сумму элементов на отрезке [l, r] (0-indexed)
        return prefix[r + 1] - prefix[l];
    }
};

int main() {
    std::vector&lt;int&gt; arr = {1, 3, 5, 7, 9, 11};
    PrefixSum ps(arr);
    
    std::cout &lt;&lt; "Sum of elements from 1 to 3: " &lt;&lt; ps.getSum(1, 3) &lt;&lt; std::endl;  // 3+5+7=15
    std::cout &lt;&lt; "Sum of elements from 0 to 5: " &lt;&lt; ps.getSum(0, 5) &lt;&lt; std::endl;  // 1+3+5+7+9+11=36
    
    return 0;
}
</code>
</pre>

<h3>Пример на C</h3>
<pre>
<code>
#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;

typedef struct {
    long long *prefix;
    int size;
} PrefixSum;

PrefixSum* createPrefixSum(int *arr, int n) {
    PrefixSum *ps = malloc(sizeof(PrefixSum));
    ps->prefix = calloc(n + 1, sizeof(long long));
    ps->size = n + 1;
    
    for (int i = 0; i < n; i++) {
        ps->prefix[i + 1] = ps->prefix[i] + arr[i];
    }
    
    return ps;
}

long long getSum(PrefixSum *ps, int l, int r) {
    // Возвращает сумму элементов на отрезке [l, r] (0-indexed)
    return ps->prefix[r + 1] - ps->prefix[l];
}

void freePrefixSum(PrefixSum *ps) {
    free(ps->prefix);
    free(ps);
}

int main() {
    int arr[] = {1, 3, 5, 7, 9, 11};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    PrefixSum *ps = createPrefixSum(arr, n);
    
    printf("Sum of elements from 1 to 3: %lld\n", getSum(ps, 1, 3));  // 3+5+7=15
    printf("Sum of elements from 0 to 5: %lld\n", getSum(ps, 0, 5));  // 1+3+5+7+9+11=36
    
    freePrefixSum(ps);
    
    return 0;
}
</code>
