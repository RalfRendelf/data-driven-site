<h2>Dynamic Programming (Динамическое программирование)</h2>

<p><strong>Категория:</strong> Оптимизация</p>

<h3>Описание</h3>
<p>Динамическое программирование - это метод решения сложных задач путем разбиения их на более простые подзадачи. Он применяется к задачам, обладающим свойствами перекрывающихся подзадач и оптимальной подструктуры. Основная идея - решить каждую подзадачу один раз и сохранить результат, чтобы избежать повторных вычислений.</p>

<h3>Принцип работы</h3>
<ol>
<li>Определить структуру оптимального решения</li>
<li>Рекурсивно определить значение оптимального решения</li>
<li>Вычислить значение оптимального решения снизу вверх (bottom-up) или сверху вниз (top-down)</li>
<li>Построить оптимальное решение по полученным результатам</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: зависит от задачи, но обычно динамическое программирование преобразует экспоненциальные алгоритмы в полиномиальные</li>
<li>Пространственная сложность: O(n) или O(n²) в зависимости от размера таблицы, необходимой для хранения промежуточных результатов</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Снижение временной сложности по сравнению с наивными рекурсивными решениями</li>
<li>Эффективное решение задач с перекрывающимися подзадачами</li>
<li>Часто позволяет находить оптимальные решения сложных задач</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует дополнительной памяти для хранения результатов подзадач</li>
<li>Не всегда очевидно, как структурировать подзадачи</li>
<li>Не применимо ко всем типам задач</li>
</ul>

<h3>Пример использования (Фибоначчи с помощью DP)</h3>
<pre>
<code>
// Вычисление чисел Фибоначчи с использованием динамического программирования

// Bottom-up подход
function fibonacciDP(n) {
  if (n <= 1) return n;
  
  const dp = new Array(n + 1);
  dp[0] = 0;
  dp[1] = 1;
  
  for (let i = 2; i <= n; i++) {
    dp[i] = dp[i - 1] + dp[i - 2];
  }
  
  return dp[n];
}

// Top-down подход с мемоизацией
function fibonacciMemo(n, memo = {}) {
  if (n in memo) return memo[n];
  if (n <= 1) return n;
  
  memo[n] = fibonacciMemo(n - 1, memo) + fibonacciMemo(n - 2, memo);
  return memo[n];
}

// Пример задачи о рюкзаке
function knapsack(weights, values, capacity) {
  const n = weights.length;
  const dp = Array(n + 1).fill(null).map(() => Array(capacity + 1).fill(0));
  
  for (let i = 1; i <= n; i++) {
    for (let w = 0; w <= capacity; w++) {
      if (weights[i - 1] <= w) {
        dp[i][w] = Math.max(
          values[i - 1] + dp[i - 1][w - weights[i - 1]], // взять предмет
          dp[i - 1][w]                                  // не брать предмет
        );
      } else {
        dp[i][w] = dp[i - 1][w]; // не можем взять предмет
      }
    }
  }
  
  return dp[n][capacity];
}
