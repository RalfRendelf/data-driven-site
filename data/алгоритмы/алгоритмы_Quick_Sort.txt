<h2>Quick Sort (Быстрая сортировка)</h2>

<p><strong>Категория:</strong> Сортировка</p>

<h3>Описание</h3>
<p>Quick Sort - это эффективный алгоритм сортировки методом "разделяй и властвуй". Алгоритм выбирает опорный элемент (pivot), затем перераспределяет массив так, чтобы все элементы меньшие опорного находились перед ним, а большие - после. Затем рекурсивно применяет тот же процесс к подмассивам.</p>

<h3>Принцип работы</h3>
<ol>
<li>Выбрать опорный элемент из массива</li>
<li>Разделить массив на два подмассива: элементы меньше опорного и элементы больше или равны опорному</li>
<li>Рекурсивно применить быструю сортировку к двум подмассивам</li>
<li>Базовый случай рекурсии: массив с 0 или 1 элементом уже отсортирован</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Лучший случай: O(n log n)</li>
<li>Средний случай: O(n log n)</li>
<li>Худший случай: O(n²)</li>
<li>Пространственная сложность: O(log n) в лучшем случае, O(n) в худшем случае</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Один из самых быстрых алгоритмов сортировки в среднем случае</li>
<li>Работает на месте (in-place), не требуя дополнительной памяти</li>
<li>Хорошая кэш-эффективность</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Плохая производительность в худшем случае (O(n²))</li>
<li>Не является стабильным (не сохраняет порядок одинаковых элементов)</li>
<li>Чувствителен к выбору опорного элемента</li>
</ul>

<h3>Пример использования</h3>
<pre>
<code>
function quickSort(arr, low = 0, high = arr.length - 1) {
  if (low < high) {
    const pivotIndex = partition(arr, low, high);
    
    quickSort(arr, low, pivotIndex - 1);
    quickSort(arr, pivotIndex + 1, high);
  }
  
  return arr;
}

function partition(arr, low, high) {
  const pivot = arr[high];
  let i = low - 1;
  
  for (let j = low; j < high; j++) {
    if (arr[j] <= pivot) {
      i++;
      [arr[i], arr[j]] = [arr[j], arr[i]];
    }
  }
  
  [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
  return i + 1;
}
