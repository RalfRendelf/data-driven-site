<h2>Sliding Window (Скользящее окно)</h2>

<p><strong>Категория:</strong> Массивы</p>

<h3>Описание</h3>
<p>Скользящее окно - это техника, используемая для решения задач на подмассивы или подстроки, когда нужно найти оптимальное решение в окне фиксированного или переменного размера. Основная идея - поддерживать "окно" (подмассив) и эффективно обновлять его границы, чтобы избежать перебора всех возможных подмассивов.</p>

<h3>Принцип работы</h3>
<ol>
<li>Использовать два указателя (left и right) для обозначения границ окна</li>
<li>Расширять окно, перемещая правый указатель</li>
<li>Когда условие нарушено, сжимать окно, перемещая левый указатель</li>
<li>Отслеживать оптимальное решение на каждом шаге</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(n), где n - длина массива</li>
<li>Пространственная сложность: O(1) или O(k) в зависимости от реализации</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Значительное улучшение производительности по сравнению с наивным O(n²) подходом</li>
<li>Минимальное использование дополнительной памяти</li>
<li>Применимость к широкому кругу задач</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует специфических условий задачи</li>
<li>Сложнее для понимания и реализации</li>
</ul>

<h3>Пример на C++: Найти максимальную сумму подмассива фиксированного размера</h3>
<pre>
<code>
#include &lt;iostream&gt;
#include &lt;vector&gt;
#include &lt;algorithm&gt;
#include &lt;climits&gt;

int maxSumSubarray(std::vector&lt;int&gt;&amp; arr, int k) {
    if (arr.empty() || k <= 0 || k > arr.size()) {
        return 0;
    }
    
    int windowSum = 0;
    
    // Вычисляем сумму для первого окна
    for (int i = 0; i < k; i++) {
        windowSum += arr[i];
    }
    
    int maxSum = windowSum;
    
    // Сдвигаем окно, добавляем новый элемент и удаляем самый левый
    for (int i = k; i < arr.size(); i++) {
        windowSum = windowSum + arr[i] - arr[i - k];
        maxSum = std::max(maxSum, windowSum);
    }
    
    return maxSum;
}

// Найти минимальную длину подмассива с суммой >= target
int minSubArrayLen(int target, std::vector&lt;int&gt;&amp; nums) {
    int n = nums.size();
    int left = 0, right = 0;
    int sum = 0;
    int minLength = INT_MAX;
    
    while (right < n) {
        sum += nums[right];  // Расширяем окно
        
        // Сокращаем окно слева, пока сумма >= target
        while (sum >= target) {
            minLength = std::min(minLength, right - left + 1);
            sum -= nums[left];
            left++;
        }
        
        right++;  // Расширяем окно справа
    }
    
    return minLength == INT_MAX ? 0 : minLength;
}

int main() {
    std::vector&lt;int&gt; arr = {2, 1, 5, 1, 3, 2};
    int k = 3;
    
    int maxSum = maxSumSubarray(arr, k);
    std::cout << "Maximum sum of subarray of size " << k << ": " << maxSum << std::endl;
    
    std::vector&lt;int&gt; nums = {2, 3, 1, 2, 4, 3};
    int target = 7;
    int result = minSubArrayLen(target, nums);
    std::cout << "Minimal length of subarray with sum >= " << target << ": " << result << std::endl;
    
    return 0;
}

<h3>Пример на C: Найти средние значения для каждого окна фиксированного размера</h3>
<pre>
<code>
#include &lt;stdio.h&gt;

void slidingWindowAverage(int arr[], int size, int k, double result[]) {
    if (size == 0 || k <= 0 || k > size) {
        return;
    }
    
    int windowSum = 0;
    
    // Сумма первого окна
    for (int i = 0; i < k; i++) {
        windowSum += arr[i];
    }
    
    result[0] = (double)windowSum / k;
    
    // Остальные окна
    for (int i = k; i < size; i++) {
        windowSum = windowSum + arr[i] - arr[i - k];
        result[i - k + 1] = (double)windowSum / k;
    }
}

int main() {
    int arr[] = {1, 3, 2, 5, 4, 6, 8, 7};
    int size = sizeof(arr) / sizeof(arr[0]);
    int k = 3;
    
    double result[size - k + 1];
    
    slidingWindowAverage(arr, size, k, result);
    
    printf("Averages of subarrays of size %d: ", k);
    for (int i = 0; i < size - k + 1; i++) {
        printf("%.2f ", result[i]);
    }
    printf("\n");
    
    return 0;
}
