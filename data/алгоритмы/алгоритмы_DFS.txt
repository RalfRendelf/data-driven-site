<h2>DFS (Depth-First Search - Поиск в глубину)</h2>

<p><strong>Категория:</strong> Графы</p>

<h3>Описание</h3>
<p>DFS (поиск в глубину) - это алгоритм обхода или поиска в графе, который начинает с корневого узла (выбирается произвольно в случае графа) и исследует как можно дальше каждую ветвь, прежде чем отступить. Алгоритм использует стек (явно или неявно через рекурсию) для отслеживания узлов, которые нужно посетить.</p>

<h3>Принцип работы</h3>
<ol>
<li>Начать с корневого узла</li>
<li>Отметить текущий узел как посещенный</li>
<li>Для каждого соседнего узла, который еще не посещен, рекурсивно вызвать DFS для этого узла</li>
<li>Продолжать до тех пор, пока все достижимые узлы не будут посещены</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(V + E), где V - количество вершин, E - количество ребер</li>
<li>Пространственная сложность: O(V) для хранения пометок о посещении и O(V) для стека вызовов в худшем случае</li>
<li>Максимальная глубина рекурсии: O(V) в случае вырожденного графа (линейного типа)</li>
<li>Средняя пространственная сложность для явной реализации стека: O(log n) для сбалансированных графов</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Малое использование памяти по сравнению с BFS</li>
<li>Подходит для поиска пути в лабиринтах</li>
<li>Используется для топологической сортировки</li>
<li>Подходит для задач на обнаружение циклов</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Может застрять в длинной ветви при глубоком графе</li>
<li>Не находит кратчайший путь в невзвешенном графе</li>
<li>Может не найти решение в бесконечной ветви (без ограничения глубины)</li>
</ul>

<h3>Пример использования</h3>
<pre>
<code>
function dfs(graph, start, visited = new Set()) {
  visited.add(start);
  console.log(start); // действия при посещении узла
  
  for (const neighbor of graph[start]) {
    if (!visited.has(neighbor)) {
      dfs(graph, neighbor, visited);
    }
  }
  
  return visited;
}

// Использование с графом, представленным списком смежности:
const graph = {
  'A': ['B', 'C'],
  'B': ['A', 'D', 'E'],
  'C': ['A', 'F'],
  'D': ['B'],
  'E': ['B', 'F'],
  'F': ['C', 'E']
};

dfs(graph, 'A');
</code>
