<h2>BFS (Breadth-First Search - Поиск в ширину)</h2>

<p><strong>Категория:</strong> Графы</p>

<h3>Описание</h3>
<p>BFS (поиск в ширину) - это алгоритм обхода или поиска в графе, который начинает с корневого узла и исследует все соседние узлы на текущем уровне, прежде чем переходить к узлам следующего уровня. Алгоритм использует очередь для отслеживания узлов, которые нужно посетить.</p>

<h3>Принцип работы</h3>
<ol>
<li>Поместить начальный узел в очередь</li>
<li>Пометить узел как посещенный</li>
<li>Извлечь узел из очереди и добавить в очередь все его непосещенные соседи</li>
<li>Повторять шаги 2-3 до тех пор, пока очередь не будет пуста</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(V + E), где V - количество вершин, E - количество ребер</li>
<li>Пространственная сложность: O(V) для хранения пометок о посещении и для очереди в худшем случае</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Находит кратчайший путь в невзвешенном графе</li>
<li>Полностью обходит компонент связности</li>
<li>Не может застрять в бесконечной ветви</li>
<li>Хорошо подходит для поиска всех узлов на заданном расстоянии</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует больше памяти по сравнению с DFS</li>
<li>Может потребовать значительного объема памяти при широком дереве</li>
<li>Медленнее в некоторых случаях из-за большого объема памяти</li>
</ul>

<h3>Пример использования</h3>
<pre>
<code>
function bfs(graph, start) {
  const queue = [start];
  const visited = new Set([start]);
  
  while (queue.length > 0) {
    const node = queue.shift();
    console.log(node); // действия при посещении узла
    
    for (const neighbor of graph[node]) {
      if (!visited.has(neighbor)) {
        visited.add(neighbor);
        queue.push(neighbor);
      }
    }
  }
  
  return visited;
}

// Использование с графом, представленным списком смежности:
const graph = {
  'A': ['B', 'C'],
  'B': ['A', 'D', 'E'],
  'C': ['A', 'F'],
  'D': ['B'],
  'E': ['B', 'F'],
  'F': ['C', 'E']
};

bfs(graph, 'A');
</code>
