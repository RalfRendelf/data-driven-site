<h2>Two Pointers: Opposite (Два указателя: навстречу друг другу)</h2>

<p><strong>Категория:</strong> Массивы</p>

<h3>Описание</h3>
<p>Алгоритм двух указателей, когда один указатель начинает с начала массива, а другой - с конца, и они движутся навстречу друг другу. Часто используется для задач на палиндромы, сортированные массивы и поиск пар с определенной суммой.</p>

<h3>Принцип работы</h3>
<ol>
<li>Инициализировать два указателя: один в начале (left), другой в конце (right) массива</li>
<li>Сравнивать элементы по указателям и выполнять требуемые действия</li>
<li>Двигать указатели навстречу друг другу в зависимости от условия задачи</li>
<li>Прекратить, когда указатели встретятся или пересекутся</li>
</ol>

<h3>Сложность алгоритма</h3>
<ul>
<li>Временная сложность: O(n), где n - длина массива</li>
<li>Пространственная сложность: O(1) - только для дополнительных переменных</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Оптимизация времени выполнения по сравнению с вложенными циклами</li>
<li>Экономия памяти по сравнению с рекурсией</li>
<li>Эффективен для сортированных массивов</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Требует отсортированных данных для некоторых задач</li>
<li>Не применим ко всем типам задач</li>
</ul>

<h3>Пример на C++: Проверка палиндрома</h3>
<pre>
<code>
#include &lt;iostream&gt;
#include &lt;string&gt;
#include &lt;cctype&gt;

bool isPalindrome(const std::string&amp; s) {
    int left = 0;
    int right = s.length() - 1;
    
    while (left < right) {
        // Пропускаем не буквенно-цифровые символы слева
        while (left < right &amp;&amp; !std::isalnum(s[left])) {
            left++;
        }
        
        // Пропускаем не буквенно-цифровые символы справа
        while (left < right &amp;&amp; !std::isalnum(s[right])) {
            right--;
        }
        
        // Сравниваем символы (в нижнем регистре)
        if (std::tolower(s[left]) != std::tolower(s[right])) {
            return false;
        }
        
        left++;
        right--;
    }
    
    return true;
}

int main() {
    std::string input = "A man, a plan, a canal: Panama";
    
    if (isPalindrome(input)) {
        std::cout << "String is a palindrome." << std::endl;
    } else {
        std::cout << "String is not a palindrome." << std::endl;
    }
    
    return 0;
}
</code>
</pre>

<h3>Пример на C: Разворот массива</h3>
<pre>
<code>
#include &lt;stdio.h&gt;

void reverseArray(int arr[], int size) {
    int left = 0;
    int right = size - 1;
    
    while (left < right) {
        // Меняем местами элементы
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;
        
        left++;
        right--;
    }
}

void printArray(int arr[], int size) {
    for (int i = 0; i < size; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int arr[] = {1, 2, 3, 4, 5, 6};
    int n = sizeof(arr) / sizeof(arr[0]);
    
    printf("Original array: ");
    printArray(arr, n);
    
    reverseArray(arr, n);
    
    printf("Reversed array: ");
    printArray(arr, n);
    
    return 0;
}
