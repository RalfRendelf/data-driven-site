<h2>C++98/C++03 (Стандарты C++98 и C++03)</h2>

<p><strong>Категория:</strong> Стандарты</p>

<h3>Описание</h3>
<p>C++98 был первым официальным стандартом языка C++, опубликованным в 1998 году. C++03 был техническим корректировочным стандартом, в основном исправляющим дефекты, опубликованным в 2003 году. Эти стандарты заложили основы языка C++ как объектно-ориентированного языка программирования.</p>

<h3>Основные особенности</h3>
<ol>
<li>Стандартизация STL (Standard Template Library)</li>
<li>Шаблоны и параметризованные типы</li>
<li>Обработка исключений</li>
<li>RTTI (Run-Time Type Information)</li>
<li>Виртуальные функции и наследование</li>
</ol>

<h3>Сложность</h3>
<ul>
<li>Временная сложность: N/A</li>
<li>Пространственная сложность: N/A</li>
</ul>

<h3>Преимущества и недостатки</h3>
<p>Преимущества:</p>
<ul>
<li>Базовый стандарт, на котором строятся все последующие версии</li>
<li>Стабильность и надежность</li>
<li>Широкая поддержка компиляторов</li>
<li>Хорошо документированная спецификация</li>
</ul>

<p>Недостатки:</p>
<ul>
<li>Отсутствие современных возможностей языка</li>
<li>Более многословный синтаксис</li>
<li>Менее безопасные конструкции по сравнению с новыми стандартами</li>
</ul>

<h4>C</h4>
<pre>
<code>
// C не поддерживает классы и шаблоны, как C++, но показывает основы
#include &lt;stdio.h&gt;
#include &lt;stdlib.h&gt;

// Структура как замена классу
typedef struct {
    int value;
} MyStruct;

// Функции для имитации методов класса
MyStruct* createMyStruct(int value) {
    MyStruct* obj = malloc(sizeof(MyStruct));
    obj->value = value;
    return obj;
}

int getValue(MyStruct* obj) {
    return obj->value;
}

void destroyMyStruct(MyStruct* obj) {
    free(obj);
}
</code>
</pre>

<h4>C++</h4>
<pre>
<code>
#include &lt;iostream&gt;
#include &lt;vector&gt;
#include &lt;algorithm&gt;
#include &lt;stdexcept&gt;

class MyClass {
private:
    int value;

public:
    // Конструктор
    explicit MyClass(int val) : value(val) {}
    
    // Геттер
    int getValue() const {
        return value;
    }
    
    // Сеттер
    void setValue(int val) {
        value = val;
    }
};

int main() {
    // STL контейнеры
    std::vector&lt;int&gt; numbers;
    
    // Заполнение вектора
    for (int i = 0; i < 10; ++i) {
        numbers.push_back(i);
    }
    
    // Алгоритмы STL
    std::reverse(numbers.begin(), numbers.end());
    
    // Обработка исключений
    try {
        MyClass obj(42);
        std::cout << "Value: " << obj.getValue() << std::endl;
    } catch (const std::exception& e) {
        std::cerr << "Exception: " << e.what() << std::endl;
    }
    
    return 0;
}
</code>
</pre>
