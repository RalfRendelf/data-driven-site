<p><strong>Цель:</strong> Сгенерируй описание страницы справки для выбранного элемента (паттерна, архитектуры, средства).</p>

<p>Формат отображения должен быть:</p>
<ul>
  <li>&lt;p&gt;&lt;strong&gt;Назначение:&lt;/strong&gt; ... &lt;/p&gt;</li>
  <li>&lt;ul&gt; для "Когда использовать"</li>
  <li>&lt;ul&gt; для "Преимущества", "Недостатки"</li>
  <li>&lt;pre&gt;&lt;code&gt;...пример кода...&lt;/code&gt;&lt;/pre&gt;</li>
</ul>

<p>Пример вызова:</p>
<pre><code>Сгенерируй HTML-файл страницы по шаблону:
1. Назначение
2. Когда использовать
3. Преимущества
4. Недостатки
5. Пример кода на C++ или Java
</code></pre>

<p>Требования:</p>
<ul>
  <li>Только HTML-теги (p, ul, pre, code)</li>
  <li>Без внешних CSS или JS</li>
  <li>Язык описания — русский</li>
</ul>
